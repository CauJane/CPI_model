Datasets:
`data_annual_input.xlsx` is an original intermediate data generated by Step 1 to 5 and is used to build the quantile regression model and perform sensitive analysis.
`data_adjust.xlsx` was used to adjust the existed model.

R codes:
`parasEstimate.R` demonstrates how to construct a model.
`sensiAnalysis.R` contains the code for sensitive analysis.
`modelAdjust.R` was applied to adjust the constructed model.
